## ✨ ویژگی‌های کلیدی

این ربات با هدف دانلود سریع و بی‌دردسر موسیقی، همراه با پنل مدیریت حرفه‌ای برای کنترل کاربران طراحی شده است.

| ویژگی | توضیحات |
| :--- | :--- |
| 🎧 **پشتیبانی از پلتفرم‌ها** | دانلود مستقیم تک‌آهنگ‌ها و پلی‌لیست‌ها با بالاترین کیفیت از **Spotify** و **SoundCloud**. |
| 📝 **دریافت متن ترانه** | یکپارچه‌سازی شده با سرویس **Genius API** برای استخراج و ارسال دقیق متن آهنگ‌ها (Lyrics). |
| 🌍 **چندزبانه (Multi-Language)** | سیستم منعطف برای تغییر زبان ربات توسط کاربر و ارائه پیام‌های متناسب با زبان انتخابی. |
| 🔒 **عضویت اجباری (Force Join)** | بررسی خودکار عضویت کاربران در کانال‌های اسپانسر پیش از ارائه خدمات به آن‌ها. |
| ⏳ **مدیریت صف (Queue System)** | پردازش بهینه پلی‌لیست‌های طولانی و هندل کردن درخواست‌های همزمان بدون افت سرعت سرور. |
| ⚡ **کش هوشمند (Smart Caching)** | جلوگیری از دانلود مجدد فایل‌ها با بررسی دیتابیس و فوروارد فوری آهنگ‌های از پیش دانلود شده. |
| 🛡️ **سیستم ضد اسپم (Rate Limit)** | اعمال محدودیت روی سرعت درخواست‌های کاربران برای جلوگیری از اسپم و کاهش فشار روی سرور. |
| 📊 **مدیریت کاربران با جنگو** | ثبت خودکار کاربران (`Members`)، ذخیره نظرات (`UserComments`) و مدیریت پیشرفته از طریق پنل ادمین Django. |

---

## 🛠️ تکنولوژی‌ها و ابزارهای استفاده شده

- **زبان برنامه‌نویسی:** Python 3.x
- **فریم‌ورک بک‌اند:** Django (برای ORM، مدل‌سازی دیتابیس و پنل مدیریت ادمین)
- **کتابخانه تلگرام:** Pyrogram / Telethon 
- **استخراج صدا:** Spotipy (برای API اسپاتیفای) و yt-dlp
- **دیتابیس:** SQLite (پیش‌فرض) / قابلیت ارتقا به PostgreSQL
